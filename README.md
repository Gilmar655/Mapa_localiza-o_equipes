## Publicação
1. Descompacte o ZIP.
2. Envie **todo o conteúdo interno** para a raiz do repositório.
3. Confirme que `index.html` e `.nojekyll` estão na raiz.
4. Em GitHub: Settings → Pages → Deploy from a branch → branch principal → `/ (root)`.
