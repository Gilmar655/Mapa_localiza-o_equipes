# Site Enel Brasil – Acompanhamento de Equipes em Campo

Site pronto para GitHub Pages.

## Como publicar
1. Descompacte esta pasta.
2. Envie `index.html`, `.nojekyll` e este `README.md` para a raiz do repositório.
3. Em Settings > Pages, selecione a branch principal e pasta `/root`.

## Correções aplicadas
- Página 100% autossuficiente em um único `index.html`.
- Relógio de data/hora em tempo real.
- Lista completa de 193 projetos visível mesmo antes do JavaScript.
- Filtros por parceira, status, tipo, data, intervenções, região e busca geral.
- Gráficos de barras coloridos para status, intervenções e tipo de serviço.
- Layout responsivo para computador e celular.
- Logomarca e foto incorporadas em Base64 para não quebrar no GitHub Pages.
