# Site de localização de equipes – Enel Brasil

Versão pronta para publicação no GitHub Pages.

## Conteúdo
- `index.html`: site principal responsivo e autossuficiente.
- `dados/projetos.csv`: base de projetos em CSV.
- `dados/projetos.json`: base de projetos em JSON.
- `dados/Converter_tabela_final.xlsx`: planilha original.
- `.nojekyll`: arquivo para evitar problemas de publicação no GitHub Pages.

## Atualizações incluídas
- Base com 193 projetos do arquivo anexado.
- Lista visível de projetos no site.
- Filtros por parceira, status, tipo de serviço, data, intervenção e busca geral.
- Gráficos de barras coloridos para status, intervenções e tipo de serviço.
- Relógio de data e hora de acesso.
- Logomarca Enel Brasil incorporada no próprio HTML.
- Foto real da equipe trabalhando na rede aérea incorporada no próprio HTML.
- Mapa Google My Maps incorporado.

## Publicação no GitHub Pages
1. Descompacte este ZIP.
2. Envie todos os arquivos para a raiz do repositório.
3. Confirme que `index.html` está na raiz.
4. Acesse Settings > Pages e publique pela branch principal.
